Introduction to buljo notes

Try to write a buljo note.


Colors

  • 'b' blue
  • 'r' red
  • 'g' green
  • 'y' yellow
  • 'a' alternative green
  • 'o' orange
  • default is black

Icons

  • '[]' square
  • '[.]' square fill
  • '[!]' exclamation road sign
  • '[^]' triangle
  • '->' right arrow
  • '()' round
  • '(x)' round check box
  • '(.)' round fill
  • ':-)' smiley
  • default is lightning
